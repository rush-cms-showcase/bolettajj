# Use Case
Build the high-impact "Home Page" using the defined "Hybrid" design strategy (Modern Body + Disruptive Soul).

# Technical Requirements
- [ ] **Hero Section:**
    - [ ] Full viewport height.
    - [ ] Video background or high-impact image.
    - [ ] Primary CTA (linked to WhatsApp or valid anchor).
    - [ ] "Manifesto" Block text.
- [ ] **Features / Feats:**
    - [ ] Grid layout using new Tailwind v4 grid utilities.
- [ ] **About / Metodologia:**
    - [ ] Asymmetric layout.
- [ ] **Reviews / Social Proof:**
    - [ ] Static mock data (until API is ready).
    - [ ] Horizontal scroll/swipe on mobile.
- [ ] **Schedule Teaser:**
    - [ ] Visual representation of the schedule (Timeline view).
- [ ] **Units / Location:**
    - [ ] Map placeholder or stylised card.
- [ ] **Contact / CTA:**
    - [ ] Final call to action before footer.

# Design Constraints
- **Performance:** Inspect Lighthouse score during dev.
- **Aesthetics:** Apply "Glassmorphism over Black Concrete" and Neon Red accents.
