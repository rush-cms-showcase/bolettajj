# Disruptive UI Concepts

> Brainstorming zone for "Boleta JJ" visual identity.

## 1. The "Choke" Transition
When navigating between pages, the screen doesn't just fade. It constricts (iris effect) or "taps out" (glitch effect).
*Feasibility:* Constructive interference with `swup` or Astro ViewTransitions.

## 2. Dynamic Typography
Headings that react to scroll velocity.
- Scroll fast -> Text skews/italicizes (Speed).
- Scroll slow -> Text stabilizes (Control).

## 3. "No-Gi" Toggle
A physical toggle switch on the UI that changes the theme from "Traditional" (White/Clean) to "No-Gi" (Dark/Neon/Rashguard patterns).
- **White Mode:** Discipline, structure, clean.
- **Dark Mode:** Chaos, submission, raw.
